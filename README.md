# 📖 Overview

ECO SHAKTI is a smart renewable energy monitoring and analytics platform designed to help users monitor, analyze, and optimize renewable energy usage efficiently.

The platform provides:

- ⚡ Real-time energy monitoring
- 📊 Smart analytics dashboards
- 🌦️ Renewable energy forecasting
- 🚨 Fault and anomaly alerts
- ☁️ Cloud-based monitoring system
- 📈 Historical performance analytics

This platform works alongside renewable energy systems and microgrid infrastructures to improve efficiency, reduce losses, and promote sustainable energy management.

---

# 🧠 Problem Statement

Renewable energy systems and microgrids often face challenges such as:

- Lack of real-time monitoring
- Delayed fault detection
- Unpredictable renewable generation
- Energy wastage and inefficiency
- Difficulty tracking long-term performance and ROI

These issues lead to:

- Increased operational costs
- Reduced efficiency
- Downtime and maintenance issues
- Poor energy optimization

---

# 💡 Proposed Solution

ECO SHAKTI provides an intelligent web-based platform that enables:

- ✅ Real-time monitoring dashboards
- ✅ Smart analytics and reporting
- ✅ Renewable energy forecasting
- ✅ Continuous anomaly detection
- ✅ Historical data tracking
- ✅ Cloud-based accessibility
- ✅ Scalable and user-friendly architecture

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Arpan-Ghosh07/ECOSHAKTI.git
```

## 2️⃣ Navigate to Project Directory

```bash
cd ECOSHAKTI
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```

## 5️⃣ Build for Production

```bash
npm run build
```

## 6️⃣ Preview Production Build

```bash
npm run preview
```
